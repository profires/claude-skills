# claude-skills

Personal skill library for Claude Code.

## Skills

| Skill | Description |
|-------|-------------|
| `edgar-api` | SEC EDGAR REST API — filings, XBRL financials, full-text search |
| `clinical-ddx` | Clinical differential diagnosis and investigation planning |

## Install

```bash
git clone https://github.com/profires/claude-skills.git ~/.claude/skills
```

Or per-project:

```bash
git clone https://github.com/profires/claude-skills.git .claude/skills
```

## Update

```bash
cd ~/.claude/skills && git pull
```
